# **Определитель матрицы**

## Задание
Реализовать вычисление определителя матрицы. \
Программа должна считывать данные из стандартного ввода и выводить результат.

### Формат входных данных
1. Первое число — размерность матрицы $n$ (матрица $n \times n$).
2. Далее $n^2$ элементов матрицы построчно или в один ряд. 

**Пример:**
- **Вход:** `2 1 0 0 1`
- **Выход:** `1`

## Технические требования

- **Ограничения:** Запрещено использовать `std::vector`
- **Типы:** Шаблонная реализация (работа с `int`, `double`, `float` и т.д. )

## Алгоритм

Вычисление определителя выполняется методом Гаусса:

1. Приведение матрицы к верхнетреугольному виду
2. Выбор главного элемента по столбцу
3. Перестановка строк при необходимости (меняет знак)
4. Умножение диагональных элементов

### Хранение данных

Для эффективного доступа к элементам матрица хранит:
- `data` — одномерный массив элементов (Continuous memory)
- `rows_ptrs` — массив указателей на начало каждой строки
