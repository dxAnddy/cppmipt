# Проект: Пересечение треугольников (3D)

Этот проект решает задачу поиска пересечений треугольников в пространстве и на плоскости.

## 📖 На основе чего сделано
Весь математический аппарат взят из книги:
**"Geometric Tools for Computer Graphics" (авторы Philip J. Schneider, David H. Eberly), страница 542.**

В данной книге описан алгоритм проверки пересечения треугольников 

## Этапы

1. **Быстрая проверка по плоскостям**: 
   Сначала смотрим, с каких сторон от плоскости первого треугольника лежат вершины второго. Если они все "сверху" или все "снизу" — значит, они точно не пересекаются. Это позволяет сразу отсеять большинство случаев.

2. **Основной случай (3D)**: 
   Если треугольники пересекают плоскости друг друга, мы находим вектор $D$ — это линия, по которой пересекаются сами плоскости. 
   Мы "схлопываем" треугольники на эту линию. Получается два отрезка. Если эти отрезки на линии накладываются друг на друга — треугольники пересекаются.

3. **Случай в одной плоскости (2D)**:
   Если плоскости параллельны и совпадают, то 3D-метод с линией не сработает. Тогда код:
   * Проецирует треугольники на самую удобную плоскость (XY, XZ или YZ).
   * Проверяет пересечение их ребер как обычных отрезков на листе бумаги.
   * Проверяет, не попала ли вершина одного треугольника внутрь другого (случай вложенности).
