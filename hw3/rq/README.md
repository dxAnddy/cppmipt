# RBTree - Красно-черное дерево с Range Queries

## Описание

Реализация красно-черного дерева с поддержкой запросов диапазона (range queries).  
Дерево поддерживает вставку элементов и подсчет количества элементов в заданном диапазоне за O(log n + k).

## Особенности

- Самобалансирующееся красно-черное дерево
- Запросы диапазона за O(log n + k)
- Итеративные алгоритмы (без рекурсии)
- Header-only библиотека

## Требования

- C++17
- CMake 3.10+
- Google Test (опционально, для тестов)

## Использование

- Вход:

```bash
k 10 k 20 q 8 31 q 6 9 k 30 k 40 q 15 40
```
- Выход:
```bash
2 0 3
```
